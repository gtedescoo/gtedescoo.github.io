The Rooting Table is a responsive single-page website created to present a collective dining and live music experience.  
The project explores the idea of slowing down, reconnecting with others, and building community through shared food and intimate live performances.

This website was developed as part of an academic web design and web development project.

---

## Project Concept

The Rooting Table is a Sunday gathering concept based in Lombardy (Italy), where people come together around one long table to share seasonal food and enjoy soft live music.

The website aims to:
- Communicate a calm and human-centred atmosphere  
- Encourage participation from guests and musicians  
- Present information in a clear, elegant, and accessible way  

---

## Technologies Used

- HTML5 for semantic structure  
- CSS for layout and visual design  
- Bootstrap 5 for responsiveness and UI components  
- Bootstrap Icons for interface elements  
- Google Fonts (Playfair Display and Nunito)  
- Vanilla JavaScript for interactions  
- Google Maps Embed for the location section  

---

## Main Features

- Fixed navigation bar with scroll-based behaviour  
- Hero section with background video and call-to-action buttons  
- Smooth scrolling navigation between sections  
- Horizontal infinite image gallery  
- Structured Sunday plan with timeline and descriptions  
- Location section with embedded map  
- Booking and musician contact via email links  
- Cookie consent banner managed with localStorage  
- Fully responsive layout for desktop and mobile devices  

---

## Visual and UX Design

The visual identity is inspired by natural and warm tones, with a palette based on sage green, terracotta, and soft cream.  
The design focuses on readability, calmness, and atmosphere, using generous spacing, rounded elements, and subtle transparency effects.

---

## Responsiveness

The website is fully responsive and adapts to different screen sizes using:
- Bootstrap grid system  
- Flexible units and media queries  
- Mobile-friendly navigation menu

## How to Run the Project
